### Prasyarat
1. Kamu punya akun blogger dan membuat blogger
Memasang template di Blogspot (Blogger) itu sebenarnya mudah, tapi ada satu hal yang paling sering bikin gagal: salah upload format file.
Berikut adalah panduan langkah demi langkah yang paling aman dan anti-gagal.

1. Persiapan (Wajib Dibaca)
Sebelum masuk ke dashboard Blogger, pastikan hal ini dulu:
Extract File: Biasanya template yang didownload berformat .zip atau .rar. Kamu harus Extract (unzip) dulu file tersebut.
Cari File .xml: Setelah di-extract, cari file yang ekstensinya .xml (contoh: template-disfo.xml). Hanya file .xml ini yang bisa diupload.

2. Cara Pasang (Metode Upload / Restore)
Ini adalah cara paling standar dan direkomendasikan.
Login ke Blogger.com.
Pilih blog yang ingin diganti templatenya (jika punya banyak blog).
Di menu sebelah kiri, klik Tema (Theme).

PENTING: Sebelum mengganti, klik tanda panah kecil (â–¼) di sebelah tombol oranye "Sesuaikan" (Customize).

Pilih Cadangkan (Backup) -> Download.
Tujuannya: Kalau template baru error, kamu bisa balikin tampilan lama dengan file backup ini.
Setelah backup aman, klik lagi tanda panah (â–¼) tadi, lalu pilih Pulihkan (Restore).

Klik Upload.
Pilih file template berekstensi .xml yang sudah kamu siapkan tadi.
Tunggu sebentar sampai muncul tulisan "Tema dipulihkan" (Theme restored).

3. Cara Alternatif (Copy-Paste HTML)
Gunakan cara ini hanya jika cara di atas gagal (biasanya muncul error "Tidak dapat memulihkan tema").
Buka file template (.xml) kamu menggunakan Notepad (di Windows) atau Text Editor lainnya.
Tekan CTRL + A (Select All) lalu CTRL + C (Copy) untuk menyalin semua kodenya.
Kembali ke menu Tema di Blogger.
Klik tanda panah (â–¼) di sebelah "Sesuaikan", pilih Edit HTML.
Di dalam editor kode, tekan CTRL + A untuk memblok semua kode lama, lalu tekan Delete atau Backspace sampai kosong.
Tekan CTRL + V (Paste) untuk menempelkan kode template baru.
Klik ikon Simpan (gambar disket ğŸ’¾) di pojok kanan atas.

4. Penyesuaian Akhir (Finishing)
Setelah template terpasang, tampilan mungkin agak berantakan karena widget lama masih "nyangkut".
Pergi ke menu Tata Letak (Layout).
Cek widget/gadget yang posisinya aneh, lalu hapus atau geser ke tempat yang sesuai.
Pergi ke menu Setelan (Settings) -> Dasar, pastikan Judul dan Deskripsi blog sudah sesuai, karena biasanya template baru menampilkannya dengan gaya berbeda
